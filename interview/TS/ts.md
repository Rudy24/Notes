### Typescript

1. Ts 和 js 的区别？
   TS 是面向对象的语言，静态类型支持模块
   js 是脚本语言，没有静态类型
   PS：什么是静态类型：在编译期间做数据检查的类型，既在写程序时要生命所有变量的数据类型

2. 什么是 TS
   TS 是 js 的类型化超集，可以编译成 js，它是面向对象的，有类，接口和像 Java 一样的静态类型语言

3. declare 作用？
   声明文件
4. 接口和类型的差别是什么？
   - 都可以描述一个对象或者函数
   - 都允许拓展（extends）

   type 可以而interface 不可以
   - type 可以声明基本类型别名，联合类型，元组等类型
   interface 可以而 type 不行
   - interface 能够声明合并
